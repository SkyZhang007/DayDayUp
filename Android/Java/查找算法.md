2. 整数中找到重复的数字
第一种，Set 集合 略
第二中，遍历数组，是数字几就放到

[0,1,2,2,3]

[1,3,3,2,4]

```
    public int findRepeatNumber(int[] nums) {
        int i = 0;
        while (i < nums.length){
            if(nums[i] == i){
                i++;
                continue;
            }
            if(nums[i] == nums[nums[i]]){
                return nums[i];
            }
            int temp = nums[i];
            nums[i] = nums[temp];
            nums[temp] = temp;
        }
        return -1;
    }
```
